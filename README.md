## Technology
- Tool: Selenium
- IDE: IntelliJ IDEA Community Edition
- Build Tool: Gradle
- Language: Java
- Test Runner: Junit

## Prerequisite
1. JDK 1.8 must be installed.
2. Requires reliable internet access

## Evaluating Locators
```sh
1. Class: $(".class") 
-> $(".form-label")	
```

```sh
2. ID: $("#id") 
-> $("#userEmail-wrapper")
```

```sh
3. Xpath: $x("") 
-> $x("//button[@id='submit']")
```

```sh
4. Attribute: $("[type=text]")
```

```sh
Note: If there is any space in the line, it should be merged by (.)
```
